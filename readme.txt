 Creativez - CSS Grid Layout Project

A modern single page web design project showcasing **CSS Grid** for flexible and responsive layouts, for my own calligraphy and art busines.

---

 Features
- Fully responsive layout using **CSS Grid**
- Clean and structured design with **minimal HTML and CSS**
- Dynamic grid system for seamless content arrangement
- Cross-browser compatibility

---

## Technologies Used
- HTML
- CSS (Grid, Flexbox, Animations)
